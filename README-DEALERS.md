# Дилерський портал — що зробити в Supabase

1. Відкрий `SQL Editor` у Supabase.
2. Створи `New query`.
3. Встав увесь код з файлу `SUPABASE-DEALERS-SETUP.sql`.
4. Натисни `Run`.

Після цього в базі з'являться:
- таблиця `dealers`
- таблиця `dealer_prices`
- view `dealer_prices_view`
- функції для входу дилера та отримання його каталогу

## Як створити першого дилера

Через адмінку сайту:
- увійди як `compadmin`
- у блоці `Дилер` додай ім'я, логін і пароль
- відкрий блок `Дилерські ціни`
- вибери дилера і задай індивідуальні ціни

## Як працює логіка цін

- назва товару, статус, роздрібна ціна — завжди беруться з таблиці `products`
- дилерська ціна — з таблиці `dealer_prices`
- якщо для товару дилерська ціна не задана, дилер бачить роздрібну ціну
