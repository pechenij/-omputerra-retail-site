# Налаштування адмінки через Supabase

Ця версія адмінки вже вміє редагувати товари прямо в Supabase.

## Один раз зроби в Supabase

1. **Створи користувача для адмінки**
   - Відкрий **Authentication -> Users**
   - Натисни **Add user**
   - Email: `compadmin@komputerra.local`
   - Password: `2V66htmPFf`
   - Познач, що email підтверджений

2. **Виконай SQL із файлу**
   - Відкрий **SQL Editor**
   - Створи новий запит
   - Встав текст із `SUPABASE-ADMIN-SETUP.sql`
   - Натисни **Run**
